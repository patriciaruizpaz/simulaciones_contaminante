Simulación de Derrame de Contaminante en un Río

Funcionalidad Principal

  • Cálculo de la Concentración:

    La concentración del contaminante se calcula utilizando la función concentracion(x, y), donde x y y son las coordenadas relativas al punto de origen. La concentración en el centro es C0 = 100 mg/L y disminuye con la distancia hasta un límite de 5 km.

  • Visualización de la Dispersión:

    Se usan círculos concéntricos coloreados para representar visualmente la dispersión desde el punto de origen. Cada círculo recibe un color basado en la concentración, calculada mediante colorConcentracion(concentracion). Los colores van de rojo a verde, indicando una alta a baja concentración.

  • Interactividad y Tooltip:

    Al mover el cursor sobre la visualización, un tooltip muestra la concentración en la posición del cursor (si está dentro de 5 km del origen). La concentración se calcula dinámicamente de acuerdo a la posición del mouse.

  • Animación del Derrame:

    La dispersión del contaminante se anima con la función animarContaminante(), que incrementa el radio del derrame hasta alcanzar el máximo (200 px o 5 km). Se utiliza requestAnimationFrame para una animación suave.

  • Indicadores de Ubicación:

    Se agregan íconos como referencia visual, incluyendo una ciudad a 5 km y un barco en movimiento sobre el borde de la zona contaminada.

Estructura del Código

El código incluye las siguientes secciones clave:

  • HTML y Estilos CSS: Define la estructura y estilo de los elementos de la página, como el fondo, la escala de concentración, pie de página, y el tooltip.
  • JavaScript con D3.js: Define las funciones para cálculo de concentración, visualización del derrame, animación y la interactividad, utilizando D3.js para manipulación SVG.

Requisitos

Para ejecutar este proyecto, es necesario:

  • Conexión a internet para cargar la biblioteca D3.js desde su CDN.
  • Un navegador moderno con soporte para HTML5 y SVG para la animación y visualización.
