Documentación: Acumulación de Contaminante en un Radio de 5 km

Este código simula visualmente la acumulación de un contaminante en un área circular de 5 km de radio desde un punto de derrame. Utiliza D3.js para crear una animación que expande visualmente la concentración del contaminante y calcula la acumulación en tiempo real.

Funcionalidades Principales

  • Visualización de Acumulación de Contaminante: Se genera un círculo que representa la zona de contaminación desde el punto de derrame, mostrando cómo la concentración del contaminante disminuye con la distancia.
  • Cálculo de la Concentración: La concentración se calcula usando la fórmula concentracion(r) = C0 / (1 + r^2), donde C0 es la concentración inicial.
  • Animación de Expansión: Mediante requestAnimationFrame, se actualiza la visualización de la acumulación con cada incremento del radio, cambiando el color de acuerdo a la concentración.
  • Escala de Concentración y Total Acumulado: La interfaz muestra una escala de colores (rojo a verde) para representar niveles de concentración, y el valor de acumulación total en tiempo real.
  • Interacción del Usuario: Al mover el cursor sobre el área de contaminación, un tooltip muestra la concentración estimada en el punto bajo el cursor.
  • Elementos Visuales: Incluye un icono de un barco en el centro que indica el punto de derrame, y una leyenda en el borde del círculo para la distancia de 5 km.

Librerías Utilizadas

Se usa D3.js para manipulación y visualización de datos mediante SVG.

Estructura del Código

  • HTML: Define la estructura de la página, la interfaz de usuario y un pie de página con información adicional.
  • CSS: Estilos para el fondo, el texto, el SVG y otros elementos visuales, incluyendo el tooltip y la escala de concentración.
  • JavaScript: Lógica para calcular la concentración, gestionar la animación de expansión y mostrar el tooltip interactivo.

Instrucciones para Visualización

Este archivo HTML puede abrirse en cualquier navegador con soporte de JavaScript. Solo se requiere conexión a Internet para cargar la librería D3.js desde el enlace proporcionado.
